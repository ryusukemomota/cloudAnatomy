# 解剖学学習ゲーム

解剖学を楽しく学べるWebベースの学習ツールです。

## 機能

### フラッシュカード
- 解剖図を見て、クリックして名前を確認
- カテゴリー選択（すべて/骨格/筋肉/神経）
- シャッフル機能
- キーボード操作対応（←→で移動、スペースでカード反転）

### クイズゲーム
- 解剖図を見て4つの選択肢から正しい名前を選択
- 全10問のクイズ形式
- 正答率の表示
- カテゴリー別の出題

## 使い方

1. `index.html` をブラウザで開く - フラッシュカードモード
2. `quiz.html` をブラウザで開く - クイズモード

各ページから相互に移動できます。

## 対応カテゴリー

- **骨格**: 21種類の骨の解剖図
- **筋肉**: 37種類の筋肉の解剖図
- **神経**: 10種類の神経の解剖図
